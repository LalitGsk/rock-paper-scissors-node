#Rock, Paper, Scissors, Node!
RPSNode is an over-the-internet implementation of rock-paper scissors written in node.js.

##Serverside:
* sanitizer
* jade
* mongoose
* mongoose-auth
* express
* socket.io
* everyauth

##Client Side:
* jQuery
* HTML5
* CSS

If I left anything out, feel free to speak up.
